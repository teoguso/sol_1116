Jupyter Notebook + Reveal.js = Beautiful Slides

A first attempt to produce a decent-looking presentation

using jupyter notebooks and reveal.js.

For all below to work, you generally need to be in the main sol_1116 folder after you do

git clone git@github.com:teoguso/sol_1116.git

or something like that.

Live presentation

You need to type

Notebook to reveal.js slides

To obtain the slides one requires to have the 'reveal.js' directory in the same directory as the notebook. Then type in bash

jupyter-nbconvert --to slides slides.ipynb --reveal-prefix=reveal.js

and view the slides in HTML format, with something like

chrome slides.slides.html

A static copy of the reveal.js repo is included here. The original repo is at https://github.com/hakimel/reveal.js/

Slides to HTML

This is a self-contained HTML version of the slides notebook. You can find it here.

Notebook-to-pdf

To use this nice nbconvert feature, just type jupyter-nbconvert --to pdf cumulant-to-pdf.ipynb this will produce a pdf-from-latex file from the notebook. Enjoy it with your viewer of choice.

Slides-to-pdf not fully working as of yet

This is a little hacky, as explained in this blogpost by @damianavila. I already included the two necessary files in the source. As also specified in the post, it seems to work only with mozilla firefox/iceweasel (do not trust the webpage you'll see, check the pdf output).
